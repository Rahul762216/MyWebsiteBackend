 # 🚀 Rahul Web Services - Website Project

 ## 📌 Project Overview

This is a **modern business website** built using:

* HTML
* CSS
* JavaScript
* Java Servlet (Backend)

It is designed for **startup services / web development business**.

---

## ✨ Features

* 🔥 Responsive modern UI
* 🎯 Hero slider with images
* 💼 Services section
* 🖼️ Portfolio gallery
* 📩 Contact form (Java Servlet backend)
* 💬 WhatsApp floating button

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Java Servlet
* Server: Apache Tomcat
* IDE: NetBeans

---

## ⚙️ How to Run (NetBeans)

1. Open project in NetBeans
2. Configure Apache Tomcat server
3. Right click project → Run

👉 Open in browser:

```
http://localhost:8080/MyWebsiteBackend/
```

---

## 📩 Contact Form Setup

Make sure servlet mapping is correct:

```java
@WebServlet("/contact")
```

And form action:

```html
<form action="contact" method="post">
``` 

## 🌍 Deployment

You can deploy this project using:

* Render (Free)
* Apache Tomcat (Manual Hosting)
